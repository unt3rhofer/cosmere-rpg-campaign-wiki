# Dodavanje novog sadrzaja
Najlakse je objasniti na primjeru.
Za kreaciju novog NPC entrya smatram da je najbolja praksa napraviti novi **sub**header (## Ime Prezime) u jednom od allies/villains/other markdown fileova. Na svaki taj header se moze linkati, npr. `[Terelin Atanam](../NPCs/other.md#terelin-atanam)`. Isto vrijedi i za nove iteme.

Medutim, za lokacije je vjerojatno preglednije napraviti odvojen .md file za pojedinu drzavu, a onda unutar tog markdown filea ponovno kreirati potrebne subheadere za naselja i tome slicne stvari.

Mislim da je tako bolje zbog nacina na koji Material tema slaze table of contents i overview (ono na lijevoj strani stranice)
